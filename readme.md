# Krivvton

Single-page marketing site for **Krivvton — Door Sealing & Acoustic Solutions**.

## Stack
Pure HTML, CSS and vanilla JavaScript — no build step required.

## Run locally
Open `index.html` directly in a browser, or serve the directory:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Files
- `index.html` — page markup and content
- `styles.css` — design system, layout, animations
- `script.js` — interactive behaviours (nav, counters, reveals, form)
